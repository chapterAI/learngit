# learn git 

###一.Git版本控制系统的优势
	1. 集中式版本控制只有中心服务器拥有一份代码，而分布式版本控制每个人的电脑上就有一份完整的代码。
	2. 集中式版本控制有安全性问题，当中心服务器挂了所有人都没办法工作了。
	3. 集中式版本控制需要连网才能工作,而分布式版本控制不需要连网就能工作。
	4. 分布式版本控制新建分支、合并分支操作速度非常快(建立指针)，而集中式版本控制新建一个分支相当于复制一份完整代码。
